# Motel la Cheminée

Site web moderne pour le Motel la Cheminée à St-Jérôme, Laurentides.

## Caractéristiques

- Design moderne et responsive
- Diaporama de photos en page d'accueil
- Présentation des chambres et forfaits
- Galerie d'installations
- Informations de contact et localisation
- Optimisé pour tous les appareils

## Technologies

- Vite
- Vanilla JavaScript
- CSS3 moderne
- Google Fonts (Montserrat & Playfair Display)

## Développement

```bash
# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev

# Construire pour la production
npm run build

# Prévisualiser la version de production
npm run preview
```

## Déploiement GitHub Pages

Ce projet est configuré pour un déploiement automatique sur GitHub Pages:

1. Pousser vers la branche `main`
2. GitHub Actions construira et déploiera automatiquement
3. Le site sera disponible à: `https://[username].github.io/motel-lacheminee/`
